# EC0249 - Impartición de cursos de formación del capital humano de manera presencial grupal

## Overview

This directory contains all reference documentation and analysis for the CONOCER competency standard EC0249.

**Standard Details:**
- **Code:** EC0249
- **Title:** Impartición de cursos de formación del capital humano de manera presencial grupal
- **Version:** 1.0
- **Competency Level:** 3
- **Sector:** Formación y Capacitación

## Standard Elements

EC0249 consists of three main competency elements:

1. **Element 1:** Preparar la sesión de capacitación
2. **Element 2:** Conducir la sesión de capacitación
3. **Element 3:** Evaluar el aprendizaje

Each element has specific:
- Performance criteria
- Knowledge requirements
- Evidence products
- Attitudes/habits
